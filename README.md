# Snyk Practice Lab

This is a small practice repository created to learn how Snyk imports a GitHub repo, scans dependencies, reviews security findings, and presents remediation guidance.

## What this lab includes

- A small Node.js app
- Open source dependency examples
- A Dockerfile for container scanning context
- A Terraform example for IaC scanning context
- Notes for explaining scan results from a Technical Success Manager perspective

## What I am practicing

- Importing a GitHub repo into Snyk
- Reviewing projects and findings
- Understanding severity and fix guidance
- Explaining direct and transitive dependency issues
- Thinking about how a customer should prioritize remediation
- Connecting scan results to DevSecOps adoption
